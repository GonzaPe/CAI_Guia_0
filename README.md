# CAI - Guía 0

Práctica grupal de la Guía 0 de Construcción de Aplicaciones Informáticas.

## Estructura

- `docs/`: apuntes y enunciados
- `src/`: un proyecto por ejercicio
- `CAI_Guia0.sln`: solución principal

## Convención
- Un proyecto por ejercicio
- Una rama por ejercicio o mejora
- Commits chicos y descriptivos
